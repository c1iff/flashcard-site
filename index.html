Epicodus Flashcards

JavaScript

A high-level, dynamic, untyped, and interpreted programming language. It has been standardized in the ECMAScript language specification.

Operator

Constructs which behave generally like functions, but which differ syntactically or semantically from usual functions.

Variables

Containers for storing data values.

Variable Naming

Use camelCase for identifier names (variables and functions). All names start with a letter.

Functions

A "subprogram" that can be called by code external (or internal in the case of recursion) to the function.

Methods

Actions that can be performed on objects.

Arguments

The real values passed to (and received by) the function.

Parameters

The names listed in the function definition.

Return

Ends function execution and specifies a value to be returned to the function caller.

Chaining methods

A technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Strings

string stores a series of characters like "John Doe".

Booleans

represents one of two values: true or false.

Undefined

indicates that a variable has not been assigned a value.

NaN

Not a Number

Escape

Character that turns special characters into string characters.

Alert

Method that displays an alert box with a specified message and an OK button.

Comments

A programmer-readable explanation or annotation in the source code of a computer program.
